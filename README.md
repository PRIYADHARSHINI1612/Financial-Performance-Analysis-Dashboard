
# Financial Performance Analysis Dashboard

## Project Overview
This project is an interactive Power BI dashboard designed to analyze financial performance using income, expense, balance, and cash flow data.

## Features
- Executive Dashboard
- Expense Analysis Dashboard
- Income Analysis Dashboard
- Financial Summary Dashboard
- KPI Cards
- Trend Analysis
- Expense Distribution
- Income Distribution
- Interactive Filters

## Tools Used
- Power BI Desktop
- DAX
- Data Modeling
- Data Visualization

- Total Expenses: 585M
- Net Cash Flow: 185M
